# Git Data Sync

Tento skill pomaha verzovat a synchronizovat datove exporty pres git, kdyz potrebujes prenaset data mezi vice pocitaci nebo je drzet pod kontrolou vedle projektu. Hodi se pro CSV exporty, zalohy SQLite a nasledne slouceni nebo obnovu dat na serveru.

Neni to nahrada databaze, ale provozni rutina pro situace, kdy data musi cestovat spolu s projektem a maji zustat dohledatelna a rozumne mergeovatelna.
