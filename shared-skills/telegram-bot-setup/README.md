# Telegram Bot Setup

Tento skill je rychly pruvodce pro zalozeni noveho Telegram bota, ziskani tokenu a nastaveni Chat ID. Pouziva se pri startu novych projektu nebo kdyz chybi zakladni Telegram konfigurace pro notifikace a ovladani.

Jeho vyhoda je v tom, ze shrnuje provozni kroky, ktere by jinak clovek delal rucne pres BotFather a testovaci zpravy. Neni o implementaci bota, ale o jeho korektnim nastaveni.
