# Security Review

Tento skill dela bezpecnostni audit kodu, hlavne kolem API klicu, autentizace, vstupu od uzivatele, deploye a dalsich mist, kde muze byt realne riziko zneuziti. Hodí se pred nasazenim, pri praci s tajnymi hodnotami nebo kdyz uzivatel vyslovne zmini bezpecnost.

Nesoustredi se na kosmetiku, ale na slaba mista, zneuzitelne chovani a chybejici ochrany. Je to prakticky bezpečnostni filtr pred release.
