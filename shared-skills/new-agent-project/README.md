# New Agent Project

Tento skill je scaffold pro novy Python AI agent projekt. Pouziva se ve chvili, kdy chces zalozit novou agentni aplikaci s rozumnou strukturou, zakladnimi soubory a workflow, ktere je rovnou pripravene k dalsimu vyvoji.

Jeho cil je zrychlit start a udrzet projekt od zacatku cisty a nasaditelny. Neni to jednorazovy snippet, ale startovni kostra pro dalsi konkretni implementaci.
