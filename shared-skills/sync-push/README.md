# Sync Push

Tento skill nahraje zmeny z repozitare `ai-prompts-and-skills` na GitHub a tim zpristupni nove prompty, skilly a instrukce i pro dalsi stroje. Pouziva se jako osobni synchronizacni krok po upravach sdilene konfigurace.

Stejne jako u `sync-pull` se cesta nema vazat na konkretni jmeno uzivatele. Skill je jeden pro vsechna prostredi a ma vychazet z home directory aktualniho uzivatele.
