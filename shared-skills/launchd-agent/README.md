# Launchd Agent

Tento skill vysvetluje, jak spravovat dlouho bezici Python agenty pres `launchd` na macOS, tedy start, stop, restart, reinstalaci plistu a overeni stavu. Hodí se, kdyz aplikace bezi jako sluzba na Macu a potrebujes ji ovladat bez rucniho spousteni z terminalu.

Dulezita cast skillu je provozni disciplina: po zmene plistu ho znovu nahrat, hlidat dvojite instance a rozlisit mezi beznym stopem a restart policy launchd.
