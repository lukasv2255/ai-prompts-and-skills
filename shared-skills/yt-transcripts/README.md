# YT Transcripts

Tento skill slouzi pro hromadne stahovani YouTube transkriptu z jednoho nebo vice kanalu, vcetne watchdogu a obnovy po rate limitu. Pouziva se, kdyz chces ziskat textovy obsah videi ve velkem a potrebujes, aby proces bezel spolehlive i dele nez jednu session.

Krome samotneho downloadu resi i provozni rezim: VPN, cookies, progress tracking, restart po chybe a rychle overeni stavu. Je to zakladni vstupni krok pro dalsi praci s YouTube obsahem.
