# YT GPT Builder

Tento skill bere stazene YouTube transkripty, tridi je tematicky a pripravuje soubory pro OpenAI Assistant nebo custom GPT. Pouziva se, kdyz chces z velkeho mnozstvi videi postavit knowledge base rozdelenou podle temat.

Nejde jen o klasifikaci, ale o celou pripravu materialu do tvaru, ktery se da rozumne nahrat a pozdeji dotazovat. Je to pipeline mezi transkripty a hotovou znalostni vrstvou.
