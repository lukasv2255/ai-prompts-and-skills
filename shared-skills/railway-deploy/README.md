# Railway Deploy

Tento skill je zakladni deploy workflow pro projekty nasazovane na Railway. Pomaha s pripravenim CLI, nastavenim env promennych, deployem, logy a rychlou diagnostikou, kdyz build nebo start neprojde.

Je vhodny pro obecne nasazovani na Railway, kdy projekt jeste nema vlastni uzce projektovy deploy skill. Bere Railway jako platformu a soustredi se na opakovatelny provozni postup.
