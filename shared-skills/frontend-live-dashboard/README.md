# Frontend Live Dashboard

Tento skill je jednoduchy vzor pro HTML a JavaScript dashboard, ktery si sam dotahuje aktualni data bez reloadu stranky. Pouziva se tam, kde chces rychle zobrazit stav API, tabulku nebo live prehled bez Reactu a bez npm toolingu.

Zameruje se na polling, loading state, error stav a prehledny render. Je vhodny jako lehka prezentacni vrstva k FastAPI nebo jinemu REST backendu.
