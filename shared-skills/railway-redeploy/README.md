# Railway Redeploy

Tento skill je kratky operacni postup pro vynuceny redeploy projektu na Railway. Pouziva se, kdyz zmena na Railway neprobehla spravne, build cache zlobí nebo je potreba sluzbu jednoduse pretocit bez slozitejsi analyzy.

Jeho smyslem je rychle a bezpecne zopakovat nasazeni a hned zkontrolovat, jestli se nova verze opravdu propsala. Dobre funguje jako doplnek k obecnemu deploy skillu.
