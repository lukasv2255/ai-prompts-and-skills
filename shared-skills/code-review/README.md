# Code Review

Tento skill je detailni code review zamereny hlavne na spravnost, rizika, udrzovatelnost a testovaci mezery. Hodi se pred commitem, pred mergem i ve chvili, kdy uzivatel chce jen nezavisly pohled na kvalitu implementace.

Neslouzi primarne k prepisovani kodu nebo refactoru; jeho hlavni uloha je najit problemy, sepsat je podle zavaznosti a rychle rict, co je potreba opravit nebo znovu overit.
