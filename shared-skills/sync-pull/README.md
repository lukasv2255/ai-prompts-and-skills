# Sync Pull

Tento skill stahuje aktualni stav repozitare `ai-prompts-and-skills` a tim synchronizuje sdilene prompty, skilly a globalni instrukce v aktualnim prostredi. Pouziva se jako osobni provozni rutina, kdy chces dostat nejnovější konfiguraci na libovolny stroj bez rozdeleni na Mac a PC.

Cesta se ma odvozovat z home directory, ne z pevneho jmena uzivatele. Skill je tedy stejny pro obe prostredi; meni se jen to, kde ma konkretni uzivatel domovskou slozku.
