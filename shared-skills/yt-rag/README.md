# YT RAG

Tento skill stavi knowledge base z YouTube kanalu tak, aby se na obsah videi slo ptat pres vyhledavani v transkriptech a nasledne shrnovani odpovedi. Pouziva se, kdyz chces dostat obsah kanalu do formy, se kterou jde pracovat jako s RAG systemem nebo jinym AI dotazovacim rozhranim.

Propojuje pripravu transkriptu, jejich prevod do dokumentu, embeddingy a nasledne rozhrani pro dotazy. MCP nebo chat integrace je jen jedna mozna vrstva navic; jadro skillu je obecna RAG pipeline nad YouTube obsahem.
